# Accessibility Quick Start Guide

Welcome! Wiki Notebook has built-in accessibility features designed to work with your needs. Here's how to use them.

## Accessibility Toolbar

Look for the **Accessibility** controls at the very top of the page, right below the main header.

### Controls Available

#### 1. 📏 Text Size
- **Default:** Normal size (100%)
- **Large:** Larger text (112.5%) — easier to read
- **Extra Large:** Even larger (125%) — best for low vision

**Tip:** All content scales proportionally, including buttons and form fields.

#### 2. 🎨 High Contrast
- **OFF:** Normal colors (default)
- **ON:** White background with pure black text

**When to use:** If you have low vision, sensitivity to color, or use a dark room.
**Note:** This automatically adjusts colors for maximum visibility while keeping the brand design.

#### 3. 🔤 Legible Font
- **OFF:** Normal font (default)
- **ON:** Atkinson Hyperlegible font

**When to use:** If you have dyslexia or find the default font hard to read.
**Why it helps:** This font has distinctive letterforms that reduce letter confusion:
- "b" looks different from "d"
- "1" looks different from "l" and "I"
- "0" (zero) is different from "O" (oh)

#### 4. 🎬 Reduce Motion
- **OFF:** Normal animations (default)
- **ON:** No animations or transitions

**When to use:** If motion causes dizziness, nausea, or distraction.
**What changes:** Buttons appear instantly instead of fading, scrolling jumps instead of smoothing.

## Keyboard Shortcuts

Don't have a mouse? Everything works with the keyboard!

### Basic Navigation
- **Tab** — Move to the next button or field
- **Shift+Tab** — Move to the previous button or field
- **Enter** — Click a button or submit a form
- **Space** — Check a checkbox or toggle

### Special Shortcuts
- **/** — Jump to the search box (when not typing)
- **Ctrl+Enter** — Save the current note (when editing)
- **Tab** (first time) — Show the "Skip to Main Content" link

## Quick Tips

### For Screen Reader Users
1. The "Skip to Main Content" link appears when you press Tab (even though you can't see it)
2. All buttons and links are labeled with their purpose
3. Dynamic updates (like selection counts) are announced
4. Categories and sections have proper headings

### For Keyboard-Only Users
1. Every button and link can be reached with Tab
2. The focus indicator (blue outline with yellow glow) shows where you are
3. You can navigate categories, edit notes, and save using just the keyboard
4. Form controls are in a logical order

### For Magnification Users
1. Text can be enlarged up to 200% with the Text Size control
2. The layout responds to larger text without breaking
3. Use your screen magnifier + Text Size for up to 2x+ magnification

### For Mobile Users
1. All accessibility features work on tablets and phones
2. Swipe with VoiceOver (iOS) or TalkBack (Android) to navigate
3. Voice control works with all interactive elements

## Saving Your Preferences

Your accessibility settings are **saved automatically** to your device. They'll be remembered the next time you visit, even if you close and reopen the app.

**Note:** Each browser profile has separate settings. If you use Chrome, Firefox, and Safari, each has its own preferences.

## Testing With Assistive Technology

### Screen Readers

**Mac/iOS Users:**
1. Press **Cmd+F5** to enable VoiceOver
2. Use **VO+Right Arrow** to navigate

**Windows Users:**
1. Download [NVDA](https://www.nvaccess.org/) (free and open-source)
2. Press **Insert+Right Arrow** to navigate

**Android Users:**
1. Open Settings → Accessibility → TalkBack
2. Swipe right to navigate forward, left to navigate backward

### Keyboard Navigation Only
1. Unplug your mouse or touch pad
2. Use Tab and Shift+Tab to navigate
3. Press Enter to click buttons
4. You'll see the blue focus ring around elements

### High Contrast Mode
**Windows:** Settings → Ease of Access → High Contrast
**Mac:** System Preferences → Accessibility → Display → Increase Contrast

### Text Magnification
Use your OS magnifier:
- **Windows:** Windows Key + Plus (+)
- **Mac:** Cmd + Plus (+)
- **Plus the Text Size control in Wiki Notebook for extra magnification**

## Still Having Issues?

### Buttons Are Hard to Click
- Try increasing **Text Size** — buttons will be bigger
- Or use keyboard navigation with Tab

### Can't Read the Text
- Try **High Contrast** for more visibility
- Or increase **Text Size**
- Or do both!

### Motion Makes You Dizzy
- Turn on **Reduce Motion** to stop all animations

### Having Trouble Reading
- Try **Legible Font** for better letterform distinction

## Accessibility Features Summary

| Feature | What It Does | When to Use |
|---------|-------------|-----------|
| **Text Size** | Make everything bigger | Hard to read text |
| **High Contrast** | White background, black text | Low vision, contrast sensitivity |
| **Legible Font** | Special font for dyslexia | Dyslexia, letter confusion |
| **Reduce Motion** | Stop all animations | Motion sensitivity, vestibular issues |
| **Keyboard Navigation** | Use Tab and Ctrl+Enter | No mouse, hand injuries, preference |
| **Screen Reader** | Hear the content read aloud | Blindness, low vision, learning disabilities |
